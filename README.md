# DevBlog

Блог-платформа для разработчиков на React + TypeScript.

## 🔗 Live Demo

[https://devblog-omega-ruby.vercel.app/](https://devblog-omega-ruby.vercel.app/)

## 🛠 Стек технологий

- React 19 + TypeScript
- Vite
- React Router v7
- Context API
- CSS Modules + CSS-переменные
- DEV.to API
- localStorage
- Vitest + React Testing Library

## 🚀 Запуск

```bash
git clone https://github.com/zexqos/devblog.git
cd devblog
npm install
npm run dev
```

## ✅ Реализованные функции

- Лента статей с DEV.to API
- Фильтрация по тегам
- Поиск с debounce
- Полный текст поста с Markdown
- Закладки и лайки (localStorage)
- Тёмная тема
- Skeleton-загрузка
- Пагинация
- 26+ тестов

## 🧪 Тесты

```bash
npm run test:run
```
